Executive summary

MathFan is a local-first React/Vite/TypeScript PWA for adaptive elementary math practice. The app stores student state in IndexedDB through Dexie, uses FSRS for spaced review, optionally syncs a snapshot to Google Drive appDataFolder, and offers optional Gemini-powered tutoring.

Local-firstPWAFSRSEvent logOptional AI

Main evidence: package.json, README, App.tsx, dexie.ts, scheduler.ts, fsrsAdapter.ts, snapshot.ts, driveSync.ts, gemini.ts.

Architecture style

Frontend shellsrc/App.tsx

Domain featuressrc/features/*

Practice, scheduler, curriculum, learning, mastery, diagnosis, sync, AI, audio, stats, dashboard.

Persistencesrc/db/*

Dexie tables and repository wrappers isolate IndexedDB access.

Generated learning itemssrc/features/curriculum/*

Deterministic item IDs make sync, rebuild, stats, and focused practice possible.
